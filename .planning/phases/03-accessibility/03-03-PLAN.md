## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| stream lifecycle → live regions | Callback timing must write to the right live region at the right moment |
| UI state → accessible name | The send button label must match the actual action to avoid misleading AT users |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-07 | Information Disclosure | Missing progress announcements for streamed responses | mitigate | Write a start message to the static polite status region when streaming begins |
| T-03-08 | Spoofing | Send button name that does not match the active action | mitigate | Update the button aria-label in `setStreamMode()` so the announced action matches the visible mode |
| T-03-09 | Information Disclosure | Errors that are visible only in toast UI | mitigate | Write failures to the static assertive alert region so screen readers receive the same error signal |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No package installs or build-step changes are part of this phase |
</threat_model>
